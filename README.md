# clamp-range

[![npm version](https://img.shields.io/npm/v/clamp-range.svg)](https://www.npmjs.com/package/clamp-range)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Numeric utilities — clamp, random, range, lerp, percentage

## Installation

```bash
npm install clamp-range
```

## Functions

- `clamp` — Clamp number between min and max
- `randomInt` — Random integer between min and max (inclusive)
- `range` — Generate array of numbers from start to end
- `lerp` — Linear interpolation between two values
- `percentage` — Calculate percentage with precision
